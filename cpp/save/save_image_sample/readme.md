# [C++] OpenCV3 描画サンプル: 画像の保存

画像を読み込み、リサイズを行ってファイルに保存します。

## 0.内容

* cv::imread - 画像の読み込み
* cv::resize - リサイズ
* cv::imwrite - 画像の保存

## 1.[サンプルコード](./main.cpp)

## 2.ビルド

``` bash
cmake .
make
```

## 3.実行

``` bash
./put_text_sample
```
